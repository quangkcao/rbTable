        table_border        :[1,'solid','#99cccc'],  
        header_bg_color     :['transparent','#6699cc'],
        header_text_color   :['#6699cc','#fff',],
        row_bg              :['none','#99cccc','#6699cc'],
        text_color          :['#6699cc','#6699cc','#FFF'],
        row_border        :[1,'solid','#99cccc'], 
        pointer_box_border  :[3,'solid','#ffcc33'],

Options & Methods

init
method : void
Option  Type  Default Value Description
table String table class of table
table_border Array [1, 'solid', '#ccc'] the border style of table [border-width, border-style, border-color]
header String th class of header in the table
header_bg_color Array ['transparent', '#ccc'] background of header [normal state,
hover state]
header_text_color String ['', ''] text color in header [normal state, hover state]
row String tr class of row that is applied
row_bg Array ['none', '#1785E6', '#336699'] background of row [normal state, hover state, click state]
text_color Array ['', '', ''] text color in row [normal state, hover state, click state]
pointer_box_border Array [1, 'dashed', '#FFF'] style for pointer box indicated when user drag mouse over the table [border-width, border-style, border-color]
sort_indicator Array ['asc', 'desc'] class identify icon for asc/desc order [asc class, desc class]
pagination Array ['', 'right', 'both'] style for paging system [limit per page, position 'left' 'right' 'center' of pagination, show both 'top' and 'bottom' of table]

selected_rows
method : Array of selected rows
Option  Type  Default Value Description
attr String name attribute of row, for example, 'name' or 'id'. if attr='html' return HTML content
